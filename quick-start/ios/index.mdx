# Quick start: iOS

1.	Define the spec sources and add Huma dependencies in your Podfile:
   ```ruby
    source 'https://github.com/CocoaPods/Specs.git'
    source 'git@github.com:huma-engineering/huma-ios-specs.git'

    pod 'HumaAppKit'
    pod 'HumaHeaderWidget'
    pod 'HumaVitalWidget'
   ```

2.	Add the [HumaEnvironment.plist](/api/integration/v1/configs/android) file with the following content to your project:

    ```xml
    <?xml version="1.0" encoding="UTF-8"?>
    <!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
    <plist version="1.0">
    <dict>
        <key>url</key>
        <string>https://hp-deployment-nsbx-1107.sbx.huma.com/</string>
        <key>bucket</key>
        <string>ccp-hsp-nsbx-1107-bucket</string>
        <key>country</key>
        <string>gb</string>
    </dict>
    </plist>
    ```

3.	Import the Huma SDK in AppDelegate.swift:
    ```swift
    import UIKit
    import HumaAppKit
    import HumaAuth
    ```
4.	Initialize the SDK in in the application(_:didFinishLaunchingWithOptions:) method:
    ```swift
    HumaApp.initialize() { app in
        app.configureAuth {
            $0.initialAuthDataProvider = { Networking.shared.sdkAuthData?.authData }
        }
    }
    ```
5.	Present the screen from SDK in your view controller code:
    ```swift
    self.token = HumaApp.presentWidgetScreen(withID: "ScreenID", from: self)
    ```
