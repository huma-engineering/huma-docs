# Module keys reference

This reference provides a list of available module keys and their dot-syntax for accessing results. Each module key offers a compile-time-safe way to fetch and observe specific data types.

## Available Modules (grouped by frameworks)

> **Note**: Ensure the necessary frameworks are imported to access these module keys. For additional details on accessing module results, refer to the main [Module Access Documentation](./module_results).
