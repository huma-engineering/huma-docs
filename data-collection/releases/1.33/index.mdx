# 1.33
| Product | Version | Release Date |
|---------|---------|--------------|
| RPM    | 1.33    | 2024-12-05   |
| Virtual     | 1.7        | 2024-12-20   |
| Seha  | 1.6        | 2024-12-20   |
| HealthCare  | 1.12        | 2024-12-20   |
| Cardio  | 1.5        | 2024-12-20   |

## Oxygen Saturation Module Enhancements
- Added fields to capture supplementary oxygen intake for more comprehensive monitoring.

## User Interface Enhancements
- Improved the patient onboarding process for a smoother user experience.
- Enhanced UI/UX for Device Integration.
- Updated the iDO watch integration interface for better usability.

## Patient Profile & Onboarding
- Enabled clinicians to create patient profiles with essential information for easier onboarding.

## Questionnaire & Feedback Features
- Created a one-time use web link for patients to complete questionnaires remotely, enhancing flexibility in data collection.

## Huma Workspace
- Launched a new "Workspace" interface to configure Remote Patient Monitoring (RPM) deployments, featuring multiple customizable widgets for optimized setup.

## RPM Configuration Enhancements
- Added the ability to enable or disable RPM features via configuration settings rather than source code modifications.
- Enabled dedicated environment deployments for hospital and clinic settings.

## Code & Performance Optimization
- Implemented minor bug fixes and code optimizations to improve overall system performance.
